# cron-scheduler-service

Distributed cron with leader election

## Features

- Production-ready code
- TypeScript/Python with full type safety
- Comprehensive error handling
- Built with Zo Computer by The BookMaster

## Installation

```bash
git clone https://github.com/Retsumdk/cron-scheduler-service.git
cd cron-scheduler-service
bun install
```

## Usage

```bash
bun run src/index.ts --help
```

## Configuration

Create `config.json` (TypeScript) or `config.yaml` (Python) for custom settings.
